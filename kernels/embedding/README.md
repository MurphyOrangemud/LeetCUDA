# embedding

## 0x00 说明

包含以下内容：

- [X] embedding_f32_kernel

- [X] embedding_f32x4_pack_kernel(float4向量化，pack版本版本)
- [X] embedding_f16_kernel(fp16版本)
- [X] embedding_f16x8_kernel(fp16向量化版本)
- [X] embedding_f16x8_pack_kernel(fp16向量化，pack版本)
- [X] PyTorch bindings
